# Google Apps Script (구글 앱스크립트) 완벽 연동 가이드

본 애플리케이션은 파이어베이스(Firebase)와 구글 스프레드시트(Google Sheets)를 모두 지원하는 하이브리드 저장 구조를 갖추고 있습니다. 구글 앱스크립트(GAS) 환경에서 구동하면 **자동으로 구글 스프레드시트를 데이터베이스로 사용**하게 됩니다.

## 1. 앱스크립트 프로젝트 설정
1. [Google Apps Script](https://script.google.com/)에 접속하여 새 프로젝트를 만듭니다.
2. `Code.gs` 파일을 프로젝트 루트의 `/gas/Code.gs` 내용으로 교체합니다.
3. 구글 시트 데이터 연동을 위해 **Drive API** 서비스 권한이 필요할 수 있습니다 (실행 시 권한 승인 창이 뜹니다).

## 2. 배포용 파일(Index.html) 생성
1. AI Studio 우측 상단의 **Export** 메뉴에서 **ZIP 다운로드**를 클릭합니다.
2. 다운로드한 ZIP 파일의 압축을 푼 후, 터미널에서 `npm install` 및 `npm run build`를 실행합니다.
   - 현재 설정에 따라 `vite-plugin-singlefile`이 적용되어 `dist/index.html` 파일 하나에 모든 코드(JS, CSS)가 포함됩니다.
3. 앱스크립트 프로젝트에서 `Index.html` 파일을 새로 만들고, 빌드된 `dist/index.html`의 모든 소스 코드를 복사하여 붙여넣습니다.

## 3. 구글 스프레드시트 확인
1. 웹 앱을 배포한 후 최초로 접속하면, 당신의 구글 드라이브에 **"SchoolDashboard_DB"**라는 이름의 스프레드시트가 자동으로 생성됩니다.
2. 이 시트의 각 탭(`Settings`, `Classes`, `Events`, `Notices`)에 데이터가 실시간으로 저장됩니다.
3. 시트에서 직접 데이터를 수정해도 웹 앱에 반영됩니다 (반영까지 최대 30초 소요).

## 4. 최종 배포
1. 앱스크립트 편집기에서 **배포 > 새 배포**를 클릭합니다.
2. 유형을 **웹 앱**으로 선택합니다.
3. **액세스 권한: 모든 사용자(Anyone)**로 설정하고 배포합니다.
4. 생성된 웹 앱 URL이 학교 현황판의 주소가 됩니다.

---
**주의사항**
- **AI 일정 추출 기능**: GAS 환경에서는 서버(Node.js)가 없으므로 파일 업로드를 통한 AI 일정 추출 기능은 작동하지 않습니다. 일정은 관리자 페이지에서 직접 입력하시기 바랍니다.
- **실시간성**: 파이어베이스와 달리 구글 시트는 30초마다 데이터를 확인하므로, 관리자 페이지에서 데이터를 저장한 후 메인 화면에 반영되기까지 약간의 시간이 걸릴 수 있습니다.
